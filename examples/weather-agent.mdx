# Weather Forecast Agent

This agent provides weather forecasts using the @agentic/weather tool.

## Implementation

```typescript
export const getWeatherForecast = async ({ location, date }) => {
  const weather = new WeatherTool()
  const forecast = await weather.getForecast(location, date)

  return {
    forecast: {
      temperature: forecast.temperature,
      conditions: forecast.conditions
    }
  }
}
```
