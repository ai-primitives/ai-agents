# Wikipedia Search Agent

This agent provides Wikipedia search and article summaries.

## Implementation

```typescript
export const searchWikipedia = async ({ query, type }) => {
  const wikipedia = new WikipediaClient()

  if (type === 'search') {
    const results = await wikipedia.search({ query })
    return {
      result: {
        title: results[0].title,
        content: results[0].snippet,
        url: results[0].url
      }
    }
  }

  const summary = await wikipedia.getPageSummary({ title: query })
  return {
    result: {
      title: summary.title,
      content: summary.extract,
      url: summary.url
    }
  }
}
```

```typescript
  type: 'search'

  query: 'Artificial_intelligence',
  type: 'summary'
})
```
